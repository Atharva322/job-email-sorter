# Job Email Sorter — Gmail Chrome Extension

A Chrome Extension (Manifest V3) that automatically classifies and organizes job application emails in Gmail using pattern-based classification.

## Features

- **Auto-labeling** — Applies Gmail labels based on email content
- **5 job categories** — Accepted, Rejected, Interview Request, Applied, Pending
- **Promo filtering** — Detects and optionally archives promotional emails
- **Gmail badges** — Visual tags injected directly into your Gmail inbox
- **Dashboard popup** — Stats, acceptance rate, and recent activity log
- **Dark mode** — Toggle in the popup header
- **Configurable** — Auto-archive, star acceptances, scan interval, notifications

## Gmail Labels Created Automatically

| Label | Color |
|---|---|
| `Jobs/Applications/Sent` | Blue |
| `Jobs/Responses/Accepted` | Green |
| `Jobs/Responses/Rejected` | Red |
| `Jobs/Responses/Interview Requests` | Dark Blue |
| `Jobs/Responses/Pending` | Yellow |
| `Promotions/Auto-Filtered` | Grey |
